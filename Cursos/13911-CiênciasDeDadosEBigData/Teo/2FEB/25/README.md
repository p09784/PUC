Ciência De Dados e Big Data

Oque é ciências de dados?

Ciência de dados consiste em extrair conhecimentos úteis a partir de um conjunto de dados grandes e diversos por meio da explorações, previsões e referências.

Explorações envolve identificar padrões nas informações.

Previsões consiste em usar informações que já conhecemos para fazer suposições fundamentadas sobre valores que desejamos conhecer.

Referência envolve quantificar o grau de incerteza.

Ciência de dados é aplicações dos pensamentos centrados em dados para compreender o mundo e resolver problemas.



Ciclo de vida da ciência de dados = Como se faz ciência de dados?
